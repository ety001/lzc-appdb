FlareSolverr

Cloudflare 盾防护绕过代理服务

简介

FlareSolverr 是一个代理服务器，用于绕过 Cloudflare 和 WAF 保护。它启动一个带有 Selenium 和 ChromeDriver 的浏览器实例，解决挑战，并将 Cookie 和 User-Agent 返回给调用者。

如何使用

本服务已通过 /ap 路径暴露 API 接口。您可以通过向以下地址发送 POST 请求来使用服务：

http://flaresolverr.[您的子域名].heiyu.space:8191/v1

示例请求 (Curl)

以下命令尝试访问被 Cloudflare 保护的网站（例如 https://www.google.com，仅作示例）：

curl -L -X POST 'http://flaresolverr.[您的子域名].heiyu.space:8191/v1' \ -H 'Content-Type: application/json' \ --data-raw '{ "cmd": "request.get", "url": "https://www.google.com", "maxTimeout": 60000 }'

配置信息

当前服务配置如下：

  • Log Level: info
  • Browser Timeout: 40000ms
  • Timezone: Asia/Shanghai

环境变量说明

以下是 FlareSolverr 支持的环境变量列表，你可以通过懒猫的设置面板进行配置：

变量名 默认值 说明
LOG_LEVEL info 日志级别。使用 debug 获取更多信息。
LOG_FILE none 日志文件路径。例如：/config/flaresolver.log。
LOG_HTML false 仅用于调试。如果为 true，所有通过代理的 HTML 将在 debug 级别下记录到控制台。
CAPTCHA_SOLVER none 验证码解决方法。当遇到验证码时使用。
TZ UTC 日志和 Web 浏览器使用的时区。例如：Asia/Shanghai。
LANG none Web 浏览器使用的语言。例如：zh_CN。
BROWSER_TIMEOUT 40000 浏览器操作超时时间（毫秒）。
TEST_URL https://www.google.com 启动时请求的测试 URL，用于确认浏览器工作正常。
PROXY_URL none 上游代理 URL。例如：http://127.0.0.1:8080。
注意： 请将示例中的 [您的子域名] 替换为您实际的应用访问地址。
