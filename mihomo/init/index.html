Mihomo Clash.Meta

访问管理面板

浏览器打开 metacubexd:
http://<懒猫IP>:9090/ui/

登录时后端地址填 http://<懒猫IP>:9090 (secret 留空)

端口说明

9090 metacubexd 面板 + mihomo API (浏览器访问)
7890 mixed 代理 (HTTP / SOCKS5 自动识别)
8001 HTTP 代理
1081 SOCKS5 代理

所有端口均免懒猫网关认证, 可直接连接。

配置订阅地址

将订阅地址写入应用的「应用数据 / mihomo / subscribe_url.txt」
(每行一个, 取首个非注释行)

示例:
https://your-airport.example.com/sub?token=xxxx

每 5 分钟自动下载并热加载一次。

让其他设备走代理

局域网设备把代理指向 (任选其一):
http://<懒猫IP>:7890 (mixed)
http://<懒猫IP>:8001 (仅 http)
socks5://<懒猫IP>:1081 (仅 socks5)
